## Lot Flutter 2 — Front office principal

Ce lot ajoute les écrans fonctionnels suivants au scaffold initial :

- Navigation principale avec 5 onglets : Accueil, Transfert, Cartes, Activités, Profil.
- Module Transfert : montant, frais, taux, bénéficiaires favoris.
- Module Bénéficiaires : recherche, favoris, ajout rapide.
- Module Historique : recherche, filtres par statut, transactions groupées.
- Module Cartes : carte virtuelle, recharge, actions sécurité, historique.
- Module Fidélité : niveau, points, progression et récompenses.
- Module Promotions : offres du moment, coupons et campagnes.
- Module Profil : informations, KYC, sécurité, notifications, langue/devise.
- Module Notifications : alertes transactionnelles, sécurité et promotions.
- Module Support : FAQ, chat, signalement transaction et contact.

Tous les écrans utilisent des données fictives pour rester indépendants du backend pendant la validation UI.
